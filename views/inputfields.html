Awesome input fields

Use inputfields for every time the claimhandler needs to type information.

Place the icon where it makes sense. If it's going to be on line with the inputfield, please place it on the left.

Don't use tooltips for inputfields. Text, icon and tooltip is to much together.

If a date needs to be noted, please look at the bottom of this page for calendarfunction.

Input field: Use this for a normal input field.

<md-input-container flex>
            <label>Kontaktperson</label>
            <i class="mdi mdi-account"></i>
            <input ng-model="user.name">
            </md-input-container>

Date: Use this for, when dates needs to be noted. It has calendar.

<md-input-container flex>
            <label>Dato for skade</label>
            <i class="mdi mdi-calendar"></i>
            <input type="date" ng-model="user.submissionDate">
            </md-input-container>

To change the length of the field, you can delete "flex" after the container.
