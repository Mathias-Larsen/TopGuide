Awesome input fields

Input field: Use this for a normal input field. The text will then be the same form and color.

<md-input-container flex> <label>Kontaktperson</label> <i class="mdi mdi-account"></i> <input ng-model="user.name"> </md-input-container>

Date: Use this for, when dates needs to be noted. It has calendar.

<md-input-container flex> <label>Dato for skade</label> <i class="mdi mdi-calendar"></i> <input type="date" ng-model="user.submissionDate"> </md-input-container>

To change the length of the field, you can delete "flex" after the container.
