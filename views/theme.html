TopPro Theme

The following palettes are used for the TopPro application. Please use these and only consider editing the color theme if it is absolutely neccesary.

Add the following to your theme configuration.


$mdThemingProvider.definePalette('primary', {
            '50': '#e5ecef',
            '100': '#ccdadf',
            '200': '#99b6bf',
            '300': '#6692a0',
            '400': '#326e80',
            '500': '#004a61',
            '600': '#003b4d',
            '700': '#002c3a',
            '800': '#001d26',
            '900': '#000e13',
            'A100': '#e5ecef',
            'A200': '#ccdadf',
            'A400': '#326e80',
            'A700': '#002c3a',
            'contrastDefaultColor': 'light'
});
