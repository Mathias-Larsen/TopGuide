Buttons

Always use raised buttons

Do not use both text and icon on a button. On raised buttons, only use text.

Raised button

Use this button for activeties. Pay, save, delete, send...
This button is without the "push" effect md-no-ink, because it's better for use on a desktop.

Colored button with icon and tooltip

Use this button in toolbars. Use the tooltip so the claimhandler will have no doubt about what the button means.
This button has the "push" effect, because it should have the same color as the toolbar. Then the user can see
that the button has been pushed.
