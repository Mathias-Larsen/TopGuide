Getting started with Material Design

This is a designlibrary with the design for all elements.

First you need to get to get Angular Material Design from bower. This should

bower install angular-material --save

Put angular dependencies on the index page

<script src="bower_components/angular/angular.js"></script>
<script src="bower_components/angular-animate/angular-animate.js"></script>
<script src="bower_components/angular-aria/angular-aria.js"></script>
<script src="bower_components/angular-material/angular-material.js"></script>

Include stylesheets

<link rel="stylesheet" href="bower_components/angular-material/angular-material.css">
<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Roboto:300,400,500,700,400italic">
<link href="bower_components/mdi/css/materialdesignicons.min.css" media="all" rel="stylesheet" type="text/css"/>

Configuring your theme. Now you can use multiple crispy themes.

var app = angular.module('StarterApp', ['ngMaterial', 'ngRoute'])
    .config(function($mdThemingProvider) {
        $mdThemingProvider.theme('default')
            .primaryPalette('cyan')
            .accentPalette('light-blue');
        $mdThemingProvider.theme('altTheme')
            .primaryPalette('red')
            .accentPalette('yellow');
        $mdThemingProvider.theme('pinky')
            .primaryPalette('pink')
            .accentPalette('purple');
        $mdThemingProvider.setDefaultTheme('altTheme');
        $mdThemingProvider.alwaysWatchTheme(true);
    });
